Library of functions¶

In the library folder we provide a set of commonly used functions and linear operators that can be used to formulate problems (see section How to use it on how to formulate a problem and invoke forbes to solve it). Each of these functions is provided as a constructor taking as input all the parameters that define the function, and returning an object: this will contain all the information needed to compute the value, the gradient or the proximal mapping of the correspondent function or its conjugate.

We now describe each of these functions and their parameters. You can also access the help file of any of these function directly from MATLAB with:

help <function-name>

Distances¶

Name and parameters \(h(x)\) Default
distBall_l2(r, c, w) \(w\|x-\Pi_{\mathcal{B}_r(c)}x\|\) r=1, c=0, w=1
distBox(l, u, w) \(\sum_i w_i(x_i-\min\{u,\max\{l,x_i\}\})\) l=-inf, u=+inf, w=1
distNeg(w, u) \(\sum_i w_i(x_i-\min\{u_i,x_i\})\) w=1, u=0
distPos(w, l) \(\sum_i w_i(x_i-\max\{l_i,x_i\})\) w=1, l=0
dist2Ball_l2(r, c, w) \(\tfrac{w}{2}\|x-\Pi_{\mathcal{B}_r(c)}x\|^2\) r=1, c=0, w=1
dist2Box(l, u, w) \(\tfrac{1}{2}\sum_i w_i(x_i-\min\{u_i,\max\{l_i,x\}\})^2\) l=-inf, u=+inf, w=1
dist2Neg(w, u) \(\tfrac{1}{2}\sum_i w_i(x_i-\min\{u_i,x_i\})^2\) w=1, u=0
dist2Pos(w, l) \(\tfrac{1}{2}\sum_i w_i(x_i-\max\{l_i,x_i\})^2\) w=1, l=0

Indicator functions¶

Name and parameters \(h(x)\) Default
indBall_l2(r, c) \(\mathrm{ind}_{\{x:\ \|x-c\|\leq r\}}(x)\) r=1, c=0
indBox(l, u) \(\mathrm{ind}_{\{x:\ l\leq x\leq u\}}(x)\) l=-inf, u=+inf
indNeg(u) \(\mathrm{ind}_{\{x:\ x\leq u\}}(x)\) u=0
indPos(l) \(\mathrm{ind}_{\{x:\ x\geq l\}}(x)\) l=0

Norms and regularization functions¶

Name and parameters \(h(x)\) Default
l1Norm(r) \(r\|x\|_1\) r=1
l2Norm(r) \(r\|x\|_2\) r=1
l2NormSum(m, r) \(r \sum_{i=1}^k \|x_i\|_2,\ \mbox{where}\ x=(x_1,\ldots,x_k),\ x_i\in\mathbb{R}^m\) m=1, r=1
nuclearNorm(m, n, r) \(r \sum_i \sigma_i(X),\ \mbox{where}\ X=[x_1\cdots x_n],\ x_i\in\mathbb{R}^m\) r=1
elasticNet(m, r) \(m\|x\|_1 + \tfrac{r}{2}\|x\|_2^2\) m=1, r=1

Other functions¶

Name and parameters \(h(x)\) Default
quadratic(Q, q) \(\tfrac{1}{2}\langle Qx, x \rangle + \langle q,x\rangle\) Q=1, q=0
quadraticOverAffine(A, b, Q, q) \(\tfrac{1}{2}\langle Qx, x \rangle + \langle q,x\rangle + \mathrm{ind}_{\{x:\ Ax=b\}}(x)\) Q=1, q=0
quadLoss(w, p) \(\sum_i w_i(x_i-p_i)^2\) w=1, p=0
quadLossOverAffine(A, b, w, p) \(\sum_i w_i(x_i-p_i)^2 + \mathrm{ind}_{\{x:\ Ax=b\}}(x)\) w=1, p=0
logLoss(m) \(m \sum_i \log(1+\exp\{-x_i\})\) m=1
powabs(p, c) \(c|x|^p\) p=1, c=1
